Analysis of Factors Influencing Student Performance

This project delves into an extensive analysis of factors influencing student academic performance using data sourced from the UCI Machine Learning Repository's Student Performance dataset. This dataset comprises comprehensive information on students' demographic, social, and academic attributes collected from two Portuguese secondary schools. The primary objective is to uncover meaningful insights into the various factors that impact students' final grades (G3) and to provide actionable recommendations for educational stakeholders.

Lego Data Analysis (Study Project)

This project analyzed a Lego dataset with information on the name, year, number of parts, theme name, parent theme, ID, and licensing status. Two questions were addressed: determining the percentage of licensed sets that were Star Wars-themed and identifying the year when Star Wars was not the most popular licensed theme.
